NodeJS Passport Boilerplate

This page is demonstration of this project.

What is this project?

It is a nodejs application including:
  • login, signup with Passport (local, google, facebook, twitter and github)
  • front end pages for login, signup and profile with bootstrap
  • a well designed project structure to start your nodejs application

Technology

Server Client
Express Bootstrap
Jade jQuery
Passport underscorejs
Mongoose

Quick deploy:

$ git clone https://github.com/diki/nodejs-passport-boilerplate.git

$ npm install

create "config.js" file under config folder (see config example)

$ npm start

open localhost:3000 in your browser
